# Mapas e Resultados Finais

Esta pasta armazena as exportações visuais, mapas temáticos e gráficos gerados a partir das modelagens realizadas no QGIS para o TCC.

## 📌 Descrição das Imagens

### Mapas Finais Consolidados
* Mapa Final Consumidores.png: Mapa temático final representando a distribuição espacial e densidade dos mercados consumidores de energia analisados.
* Mapa Final Produtores.png: Mapa final detalhando a localização, potencial e infraestrutura dos produtores de energia renovável.

### Gráficos e Análises Complementares
* imagem1.png: Gráfico ou mapa auxiliar focado na primeira etapa de cruzamento de dados de oferta.
* imagem2.png: Elemento visual focado na análise de cenários de demanda de hidrogênio.
* Imagem3.png: Visualização complementar dos resultados estatísticos regionalizados.
* Imagem4.png: Mapa de apoio contendo recortes específicos ou detalhes de infraestrutura estudados.
