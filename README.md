# TCC

# Avaliação do Potencial Regional para a Implementação de Hubs de Hidrogênio no Brasil

Repositório dedicado ao armazenamento, organização e documentação das bases de dados geográficas e resultados do Trabalho de Conclusão de Curso (TCC).

# Estrutura do Repositório

O projeto está dividido em três pastas estruturais para facilitar a navegação e reprodutibilidade:

base_de_dados/: Contém planilhas consolidadas em formato `.csv` (dados da ANEEL, índices calculados por estado), projetos do QGIS (`.qgz`) e arquivos iniciais de estruturação.

camadas_geograficas/: Armazena as bases vetoriais tratadas em formato GeoPackage (`.gpkg`) e Shapefile (`.shp`), contendo polígonos de infraestrutura, dados de oferta renovável e mercado demandante.

resultados: Pasta dedicada aos mapas finais exportados do QGIS e gráficos analíticos gerados ao longo da pesquisa.
